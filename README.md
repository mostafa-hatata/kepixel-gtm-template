# KePixel GTM Template

KePixel tracker is a Google Tag Manager community template for sending website events to KePixel server-side tracking and Conversion API.

## Files

- `template.tpl` — Google Tag Manager template file
- `metadata.yaml` — Community Template Gallery metadata
- `LICENSE` — Apache 2.0 license

## Installation

1. Open Google Tag Manager.
2. Go to **Templates**.
3. Choose **New** or **Import**.
4. Import `template.tpl`.
5. Configure the required KePixel fields.
6. Test using GTM Preview Mode before publishing.

## Community Template Gallery submission

Before submitting this repository to Google Tag Manager Community Template Gallery:

1. Make sure all files are in the root of the repository.
2. Copy the Git commit SHA that contains the final `template.tpl`.
3. Add that commit SHA inside `metadata.yaml`.
4. Keep the repository public before submitting to Google.
5. Submit the GitHub repository URL from the Community Template Gallery.
